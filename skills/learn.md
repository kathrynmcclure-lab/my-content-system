# Skill: Learn

Extract and save insights from the current conversation or provided content.

## Usage

Invoke this skill to capture learnings, patterns, or decisions worth remembering.

## Instructions

1. Review the current conversation or provided content
2. Identify:
   - Key decisions made and their rationale
   - Patterns or conventions established
   - Corrections or feedback given
   - Context that will inform future work
3. Write a concise summary to `memory.md`
4. Organize by category (Decisions, Patterns, Feedback, Context)
